Chào mừng tới FITS

FIST là trang web chia sẻ về Base, Emulators và nhiều chủ đề khác sẽ được cập nhật trong tương lai. Phần mềm tự động hoá chơi, story thư viện về game trên github và hỗ trợ xuất file json cho MAA.

Hướng dẫn nhanh

  • Nhấn chữ F trên bàn phím để đóng / mở sidebar.
  • Click vào nút chữ F trong sidebar cũng được.
  • Nhấn nút Đổi nền: Trắng / Đen để chuyển theme.
