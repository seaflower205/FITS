Chào mừng tới FITS

FIST là trang web chia sẻ về Base, Emulators và nhiều chủ đề khác sẽ được cập nhật trong tương lai. Phần mềm tự động hoá chơi, story thư viện về game trên github và hỗ trợ xuất file json cho MAA.

Hướng dẫn nhanh

  • Nhấn chữ F trên bàn phím để đóng / mở sidebar.
  • Click vào nút chữ F trong sidebar cũng được.
  • Nhấn nút Đổi nền: Trắng / Đen để chuyển theme.

Maa Tool

Công cụ MAA (Maa Automation Assistant) giúp bạn tự động hoá các tác vụ trong game. Bạn có thể tạo kịch bản tự động để chơi game hoặc thực hiện các tác vụ lặp đi lặp lại.

Tính năng chính

  • Tự động hoá game thông qua các script
  • Hỗ trợ xuất file JSON
  • Giao diện trực quan và dễ sử dụng

Base

Thư viện Base chứa các tài nguyên và dữ liệu cơ bản cho các trò chơi được hỗ trợ. Đây là nền tảng cho các tính năng khác trong FITS.

Nội dung

  • Thư viện dữ liệu game
  • Cấu hình máy chủ
  • Tài liệu hướng dẫn

Emulators

Danh sách các emulator được hỗ trợ và hướng dẫn cài đặt. FITS tương thích với nhiều emulator phổ biến để chạy game trên máy tính.

Emulator được hỗ trợ

  • MuMu Emulator
  • NoxPlayer
  • BlueStacks
  • LDPlayer
